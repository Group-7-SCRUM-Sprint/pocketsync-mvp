# PocketSync

A consolidated personal finance management platform designed to help Nigerians manage multiple bank and fintech accounts from a single interface.

## Problem

Managing finances across multiple banking and fintech applications can be fragmented and time-consuming. Users often switch between platforms to check balances, track spending, and understand their overall financial health.

PocketSync simplifies this experience by bringing financial information together in one place, helping users gain visibility, control, and confidence in their money management.

## Sprint Focus

This project is being developed as part of a 5-day Scrum sprint.

### MVP Feature

**Account Aggregation Dashboard**

The MVP enables users to:

* Connect multiple financial accounts (mock integrations)
* View combined account balances
* Monitor recent transactions
* Access a unified financial overview
* Make informed financial decisions from a single dashboard
